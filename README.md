# Custom Laravel Components

Овој репозиториум содржи само custom компоненти и view-ови од Laravel/Livewire апликација.

## Што е вклучено?
- **app/Livewire/** — Сите Livewire компоненти (PHP класи)
- **resources/views/livewire/** — Blade view-ови за Livewire компонентите
- **app/Http/Controllers/** — Само custom контролери (ако има)

## Што НЕ е вклучено?
- Core Laravel фајлови (framework, vendor, config, routes, итн.)
- .env, composer.json, package.json, storage, public, database, итн.
- Овие фајлови не се доволни за самостојно стартување на апликацијата.

## Цел
Оваа структура е наменета за презентација на custom код, без да се сподели целиот проект или чувствителни податоци.
